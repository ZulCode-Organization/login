# ZulCode 🚀

Plataforma gamificada de aprendizado de programação.

## Telas

- `/login` → Login e Cadastro (com tabs)
- `/nivelamento` → Quiz de avaliação inicial (5 perguntas)
- `/home` → Roadmap com trilha de aprendizado
- `/atividade` → Desafio diário com quiz
- `/loja` → Loja de itens com ZulCoins
- `/perfil` → Perfil do usuário com stats e conquistas

## Como rodar

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Mobile-first

## Fluxo

Login → Nivelamento → Home → Atividades
